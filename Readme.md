# 🏥 Medical Waste Detection using AI

## 📌 Project Overview
This project is developed to automatically detect medical waste using Artificial Intelligence and a webcam. The system identifies items such as syringes, gloves, masks, and glucose bottles in real-time.

This helps in proper medical waste segregation and improves safety in hospitals and clinics.

---

## 🎯 Objectives
- Detect medical waste using AI
- Use webcam for real-time monitoring
- Improve waste management and safety
- Reduce human error in waste identification

---

## 🧠 Technologies Used
- Python
- OpenCV
- TensorFlow / YOLO
- VS Code

---

## 📷 Features
- Real-time detection using webcam
- Pre-trained/custom dataset support
- Identifies different types of medical waste
- Easy to use interface
